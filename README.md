# Computational Physics Simulations in GNU Octave

## 📌 Overview

This repository contains a collection of **advanced physics simulation programs** implemented in GNU Octave.
Each program corresponds to a practical module covering topics in:

* Computational Physics
* Classical Mechanics
* Statistical Mechanics
* Astrophysics & General Relativity
* Condensed Matter Physics
* High Energy Physics

The aim of this project is to combine **numerical methods, visualization, and physical modeling** into interactive simulation suites.

---

## 🚀 Key Features

* Interactive menu-driven simulation programs
* Real-time animations and physics visualization
* Numerical methods (Monte Carlo, ODE solvers, fitting)
* Data-driven physics (DFT analysis)
* Modular and well-structured code

---

## 📂 Programs Included

### 🔹 Practical 1: Monte Carlo Methods

📄 File: 

* Linear Congruential Generator (LCG)
* Monte Carlo estimation of π
* Monte Carlo numerical integration
* 2D random walk simulation

---

### 🔹 Practical 2: Solar System Simulation

📄 File: 

* Planetary orbital motion (Keplerian dynamics)
* Elliptical orbits with eccentricity
* Mercury perihelion precession (GR correction)
* Moon orbit and comet trajectory
* Interactive speed control and visualization

---

### 🔹 Practical 3: Dynamics Simulation Suite

📄 File: 

* E × B drift motion
* Circular motion with velocity & acceleration vectors
* Simple pendulum (phase space + animation)
* Projectile motion with air drag

---

### 🔹 Practical 4: Wave Interference

📄 File: 

* 2D interference of waves
* Fringe pattern visualization
* Parameter variation (λ, slit separation, screen distance)
* High-resolution intensity plots

---

### 🔹 Practical 5: Gravitational Physics

📄 File: 

* Spacetime curvature visualization
* Gravitational potential well (geometric model)
* Gravitational wave propagation (dynamic simulation)

---

### 🔹 Practical 6: Statistical Physics

📄 File: 

* Density of states (g(E))
* Maxwell–Boltzmann, Fermi–Dirac, Bose–Einstein distributions
* Temperature dependence
* Population density (f(E) = g(E)n(E))
* Numerical integration for particle count

---

### 🔹 Practical 7: Condensed Matter (DFT Analysis)

📄 File: 

* Reads DFT dataset (`DFT_data.csv`)
* Calculates:

  * Deformation potential
  * Elastic modulus
  * Effective mass
  * Carrier mobility
* Uses polynomial fitting and physical models

---

### 🔹 Practical 8: High Energy Physics (QPM)

📄 File: 

* Parton Distribution Functions (PDFs)
* Callan-Gross relation validation
* DGLAP evolution (scaling violation)
* Deep inelastic scattering concepts

---

1. Clone the repository:

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

2. Run any program in Octave:

```octave
P1
P2
P3
P4
P5
P6
P7
P8
```
